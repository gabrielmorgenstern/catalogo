# Daniel & Clarice Aquatics

Site estático em HTML, CSS e JavaScript, pronto para publicar no GitHub Pages.

## Estrutura

- `index.html`: página principal
- `assets/css/styles.css`: estilos do site
- `assets/js/script.js`: comportamento dos botões de WhatsApp
- `assets/images/`: imagens usadas no site
- `assets/icons/`: favicon, ícone Apple Touch e ícone do WhatsApp
- `assets/images/originals/`: logos originais preservadas

## Publicação no GitHub Pages

1. Suba todos os arquivos para um repositório no GitHub.
2. Em `Settings > Pages`, escolha a branch principal.
3. Em `Build and deployment`, selecione `Deploy from a branch`.
4. Em `Folder`, selecione `/root`.
5. Salve e aguarde o link do Pages ser gerado.
